## Add More Projects

새 프로젝트는 저장소 루트에 폴더를 하나 만들고, 그 안에 `index.html`을 넣으면 됩니다.

- Example: `/Users/gim-yechan/Claude-project/new-project/index.html`
- URL: `https://tioon.github.io/market_project/new-project/`

프로젝트를 추가한 뒤에는 GitHub에 push하면 Pages가 자동으로 다시 배포됩니다.
